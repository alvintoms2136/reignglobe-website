## Features

- **Responsive Design**: Pixel-perfect layouts for desktop (1440px+), tablet (768-1439px), and mobile (<768px)
- **Smooth Animations**: CSS-only animations with scroll-triggered reveals and kinetic interactions
- **Modern UI/UX**: Glass morphism, gradient meshes, and technical grid aesthetics
- **Performance Optimized**: Fast load times, Intersection Observer API, and efficient animations
- **Accessibility**: ARIA labels, semantic HTML5, and reduced motion support
- **SEO Ready**: Structured metadata and semantic markup

## Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local web server for testing (Live Server, Python SimpleHTTPServer, etc.)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/alvintoms2136/reignglobe-website.git
   cd reignglobe-website
   ```

2. **No build process required!**
   This is a static HTML page with no dependencies.

## How to Run Locally

### Option 1: Direct File Opening
Simply double-click `index.html` and it will open in your default browser.

### Option 2: Local Web Server (Recommended)

#### Using Python 3:
```bash
python -m http.server 8000
```
Then visit: `http://localhost:8000`

#### Using Node.js (http-server):
```bash
npx http-server -p 8000
```
Then visit: `http://localhost:8000`

#### Using VS Code Live Server:
1. Install the "Live Server" extension
2. Right-click `index.html`
3. Select "Open with Live Server"

## Customization

### Update Contact Information
Edit the footer section in `index.html`:
```html
<p>Email: <a href="mailto:hello@reignglobe.com">hello@reignglobe.com</a></p>
<p>Phone: +1 (555) 123-4567</p>
```

### Modify Brand Colors
Update CSS variables in the `:root` selector:
```css
:root {
    --electric-blue: #00AEEF;
    --charcoal: #1A1A1A;
    --white: #FFFFFF;
}
```

### Add Your Logo
Replace the animated bars visualization with your logo image in the navigation section.
